Об институте

Институт последипломного медицинского образования

ТОО «Imed Institute» проводит аккредитованные курсы повышения квалификации для врачей и среднего медицинского персонала. Каждый выпускник получает именной сертификат с уникальным номером и QR-кодом для проверки подлинности.

Реестр сертификатов открыт для работодателей, страховых компаний и государственных органов — подлинность документа можно подтвердить онлайн за несколько секунд.

«Развиваем компетенции сегодня — создаём медицину будущего»

Миссия института — повышение профессиональных компетенций медицинских работников посредством качественного, практико-ориентированного и современного образования.
